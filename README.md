# GreenPass (Fork)

**GreenPass** — это практически универсальный прокси-плагин для Android-клиентов Telegram (например, exteraGram), предназначенный для обхода блокировок и проксирования всего трафика приложения, включая голосовые звонки.

## Основные возможности

* **Широкая поддержка протоколов:** VLESS, VMess, Trojan, ShadowSocks, Hysteria2 (работает на базе ядра sing-box).
* **Встроенные альтернативные режимы:**
  * **TG WS** (Telegram WebSocket)
  * **AmneziaWG** (нативный WireGuard-клиент с защитой от блокировок)
  * **olcRTC** (кастомный WebRTC-протокол)
* **Проксирование звонков:** Полноценное перенаправление VoIP-трафика звонков через Relay-серверы.
* **Удобный интерфейс настроек:**
  * Объединенный экран настройки VLESS: все функции импорта, проверки пинга, обновления подписок и очистки серверов находятся на одном экране.
  * Прямой вывод списка добавленных серверов («Свои узлы») и подписок в меню настроек.
  * Быстрый переход к конфигурации активного протокола кликом по блоку статуса в главном меню настроек.
* **Умные правила:** Автоматическое отключение прокси при работе системного VPN, списки исключений для определенных домашних/рабочих Wi-Fi сетей.

## Разработка и сборка плагина

Исходный код плагина разделен на логические модули в папке `src/` для удобства разработки и версионирования:

* `src/metadata.py` — метаданные, локализация, константы.
* `src/utils.py` — CBOR-парсеры, сетевые утилиты и декодирование ссылок.
* `src/cores.py` — управление ядрами sing-box, xray, WireGuard.
* `src/hooks.py` — хуки и перехватчики API Telegram.
* `src/main.py` — основная логика плагина и методы построения UI настроек.

### Как собрать плагин

Для сборки плагина в монолитный файл `GreenPass.plugin` запустите в корневой папке проекта скрипт сборщика:

```bash
python build.py
```

Сборщик автоматически:
1. Склеит исходные файлы в правильном порядке.
2. Удалит локальные импорты разработки.
3. Подменит метаданные (автора, версию, иконку).
4. Проверит синтаксис готового файла на ошибки компиляции.
